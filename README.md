# Proyecto katas Python

Aqui dejo los ejercicios resueltos del proyecto de katas de Python. Son 41 ejercicios en total, los he ido haciendo mas o menos en orden aunque algunos me han costado bastante.

## Archivos

- katas_python.py → el archivo con todos los ejercicios
- README.md → este archivo

## Pasos que he seguido

Primero leí todos los ejercicios para ver de que iban. La mayoria usan map(), filter() y reduce() que no las habia usado mucho antes así que tuve que buscar como funcionaban.

Los primeros ejercicios (funciones básicas, listas...) fueron mas o menos bien. Lo que mas me costó fue la parte de reduce() porque no la entendía muy bien al principio, y tambien las clases del 34 y el 36.

El ejercicio 36 me da un error al ejecutarlo porque Bob no tiene suficiente saldo para la transferencia, no me di cuenta hasta el final y ya no lo he cambiado. La clase funciona bien, es el ejemplo que está mal.

La parte de excepciones tampoco la tengo del todo clara, creo que los ejercicios 8 y 11 los he dejado sin ejecutar porque pedían input del usuario y no sabia muy bien como probarlo.

Para los ejercicios de lambdas la verdad es que en algunos casos no se si era necesario usarlas o si simplemente con una funcion normal hubiera valido, pero bueno el enunciado lo pedia así.

## Como ejecutarlo

```
python katas_python.py
```

Nota: los ejercicios 8, 11, 31 y 41 piden input al usuario así que están definidos pero no se llaman desde el codigo.
